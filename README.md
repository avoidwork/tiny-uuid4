# tiny-uuid4
[![build status](https://secure.travis-ci.org/avoidwork/tiny-uuid4.svg)](http://travis-ci.org/avoidwork/tiny-uuid4)

Tiny UUID version 4 for Client and Server

## Example
```javascript
const uuid = require("tiny-uuid4");
console.log(uuid());
```
